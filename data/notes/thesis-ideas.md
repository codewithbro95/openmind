# Thesis Ideas

The strongest thesis direction is about local-first personal AI memory.

Core claim:
Personal AI assistants become more trustworthy when they can cite local source files,
keep user data on the computer, and separate retrieval from answer generation.

Possible chapters:

1. Local storage and privacy boundaries.
2. File extraction and chunking.
3. Embedding search over messy personal archives.
4. Source-grounded answers.
5. Human trust in personal AI systems.
