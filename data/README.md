# OpenMind Test Data

This folder is a small local indexing sandbox.

Try:

```bash
openmind source add ./data
openmind index start
openmind index status
openmind search "holiday plan"
openmind ask "What does my test data say about FleetPulse?"
```

Images in `assets/` are intentionally included even though v0.2 does not index image pixels yet.
